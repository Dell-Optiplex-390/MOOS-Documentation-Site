Virtualising MOOS in a Virtual Machine(VM)

If you want to run MOOS in a virtual machine read on. Using a virtual machine(VM) for MOOS does have it's up's and down's, for example MOOS in a VM can save time especially for developers, and there is a high chance of drivers and all sorts of bits and bobs to work, a downside could be it requires a tad bit of computing power and it may be a bit slow and laggy. But anyways lets start!

Recommended/Supported Virtualising Software

  • Oracle VM Virtualbox: download
  • VMware Workstation Player: download
  • VMware Workstation Pro download
  • QEMU(Recommended to be used on Linux for maximum perfomance): download

You can download any of the above, although some of the software above require a license which may cost, for example VMware.
